## 律师审查嵌入点

在审查会议上，建议按以下顺序逐层审查：

```
第 1 小时    材料规范层    案例的原始材料 → 审核结果 → 结构化数据
第 2 小时    九步法 1-5   诉请固定 → 请求权 → 抗辩 → 要件 → 主张
第 3 小时    九步法 6-9   争点 → 举证 → 事实认定 → 归入裁判
第 4 小时    综合审查     法条引用 + 输出报告 + 反馈表填写
```
